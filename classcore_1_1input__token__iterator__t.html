Detailed Description

Input token iterator.

This is (almost) an input iterator (lacks homogeneous type comparison) that represent a given token of a token_stream_t. Incrementing this iterator may trigger that its associated token_stream_t consumes characters of the istream in order to build new tokens. Moreover, when incrementing this iterator, all the previous tokens cannot be accessed anymore.
